Projekt Übersicht

WaterBan ist ein professionelles Ban Plugin für BungeeCord. Bald wird es auch eine Version für Spigot/Paper geben, aber das wird noch dauern, denn erst einmal wird die Version für die Proxy-Server fertig. :^)

Features

- Bannen und Muten von Spielern so einfach wie nie!
- Durch die Configuration File vollständig konfigurierbar.
- Durch Speicherung der Daten in einer SQL-Datenbank besonders schnell.
- Coming soon: AdminControlPanel für einfache Verwaltung und erweiterte Funktionen.
- Coming soon: AutoBan bzw. AutoFlag-Funktionen durch externe AntiCheat Systeme.
- Coming soon: Webhook für Notifications auf einem Discord-Server.
